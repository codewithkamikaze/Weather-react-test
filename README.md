# Weather App 🌤️

![Weather App Screenshot](../screenshot.png)

## Overview

A React app that shows the current weather for Aleppo. Supports English and Arabic with right-to-left layout for Arabic. Uses Material UI, Axios, Moment.js, and i18next.

## Features

- Current temperature, min/max, and description
- Weather icons from OpenWeatherMap
- Real-time date and time
- English / Arabic toggle
- Responsive design

## Tech Stack

- React
- Material UI
- Axios
- Moment.js
- i18next
- OpenWeatherMap API

## Installation

```bash
git clone https://github.com/codewithkamikaze/Weather-react-test.git
cd weather-app
npm install
```
